显示Mac的隐藏文件夹

| 阅读量 次 发表于 Jun 14 2017 | 分类于 技术分享 ， Mac |
如果找不到对应的文件，在终端执行这两条命令可以显示Mac的隐藏文件夹(false/true 用来控制是否显示隐藏文件夹)
  1. defaults write com.apple.finder AppleShowAllFiles -bool false
  2. killall Finder
